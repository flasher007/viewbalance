Запустить

```npm i```

В папке config создать файл development.json и заполнить

```
{
  "web3provider": {
    "url": "" //адрес провайдера, например infura
  },
  "etherscan": {
    "key": "" // api ключ от https://etherscan.io
  }
}
```

Запустить приложение

``` npm run ```

Открыть в браузере http://127.0.0.1:4000
